## 🚀 Projetos em Destaque
- **Arquitetura Moderna do Sistema Bancário (NestJS + Serverless)** (projeto corporativo privado) — Desenvolvimento e manutenção de sistemas em arquitetura de microsserviços, utilizando NestJS e AWS Serverless, com API Gateway e Lambda Authorizer para autenticação e segurança de APIs.
- **Sistema Bancário Completo (projeto corporativo privado)** — Criação de um sistema bancário completo, com modelagem de banco de dados, backend, frontend e landing page. O projeto envolveu integrações financeiras, análise de dados e arquitetura MVC.
- **Gerenciador de Benefícios (projeto corporativo privado)** — Desenvolvimento de um sistema para gestão de benefícios corporativos (refeição, alimentação e home office), com módulos de cadastro de funcionários e emissão de cartões, implementado com microsserviços e APIs REST.
- **Migração de Sistema Bancário (projeto corporativo privado)** — Participação na migração de um sistema legado em Go para uma arquitetura hexagonal em TypeScript.
- **Sustentação de Sistema Bancário (projeto corporativo privado)** — Atuação na manutenção e evolução de um sistema de conta digital, com funcionalidades de Pix, boletos, extrato e cartões, realizando análise de logs, correção de incidentes e desenvolvimento de novas features.
- **Aplicativo Flutter (projeto corporativo privado)** — Desenvolvimento de um aplicativo mobile em Flutter, reforçando o domínio em desenvolvimento multiplataforma e boas práticas de UX/UI.
